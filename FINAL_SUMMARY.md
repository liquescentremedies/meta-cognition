## 🎮 How to Use

### Keyboard Controls
| Key | Action |

| `M` | Toggle reduced motion |
| `?` or `H` | Show help |
| `Space` | **Easter egg** — reveal hidden patterns |

### Mouse
- **Hover**: Preview card appears
- **Click**: Zoom into world / show details
- **Drag**: Rotate view
- **Scroll**: Zoom in/out
